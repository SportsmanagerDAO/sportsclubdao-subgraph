# Subgraph for SportsClubDAO

## Steps to Run

Run the `prepare` script, in this case, use Goerli.

```
yarn prepare:goerli
```

Generate the needed code

```
yarn codegen
```

Build the GraphQL subsgraph.

```
yarn build
```
